## ğŸ” Key Concepts Covered

### Hacking
Hacking is the process of exploiting system vulnerabilities to gain unauthorized access by modifying system or application behavior beyond its original purpose.

### Ethical Hacking
Ethical Hacking involves **authorized testing** of systems using attacker techniques to identify vulnerabilities and improve security posture.

### Hacker vs Ethical Hacker
- **Hacker:** Explores systems to find exploitable weaknesses.
- **Ethical Hacker:** Identifies vulnerabilities **with permission**, without causing harm, on behalf of system owners.
